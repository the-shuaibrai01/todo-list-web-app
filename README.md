# ✅ To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.

This project helps users manage daily tasks by allowing them to add, complete, and organize tasks efficiently.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive user interface
- Clean and simple design
- Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
